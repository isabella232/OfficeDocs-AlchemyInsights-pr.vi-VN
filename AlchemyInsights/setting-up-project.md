# <a name="setting-up-project"></a>Đang thiết lập Project

 Để thiết lập Microsoft Project, hãy làm theo các bước trong [bài viết này](https://support.office.com/article/7059249b-d9fe-4d61-ab96-5c5bf435f281.aspx).

**Thông báo cho người dùng máy Mac:** Sản Project này hiện không khả dụng cho MacOS. 
  
Nếu bạn gặp phải lỗi trong quá trình thiết lập, [hãy Project cài đặt bằng cách sử dụng Công cụ Trợ giúp Phục hồi và Hỗ trợ.](https://aka.ms/SaRA-ProjectSetupScenario)
