# <a name="single-sign-on-sso-in-microsoft-edge"></a>Bật đăng nhập một lần (SSO) trong Microsoft Edge

Microsoft Edge hỗ trợ các loại đăng nhập đơn (SSO) sau đây:
- SSO dựa trên PRT
- Seamless SSO
- SSO dựa trên WIA
